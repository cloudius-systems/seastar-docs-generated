Memory management

Detailed Description

Functions and classes for managing memory.

Memory management in seastar consists of the following:

  • Low-level memory management in the memory namespace.
  • Various smart pointers: shared_ptr, lw_shared_ptr, and foreign_ptr.
  • zero-copy support: temporary_buffer and deleter.
