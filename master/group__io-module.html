using priority_class_ptr = lw_shared_ptr<priority_class>
related

Priority class, to be used with a given fair_queue.

An instance of this class is associated with a given fair_queue. When registering a class, the caller will receive a lw_shared_ptr to an object of this class. All its methods are private, so the only thing the caller is expected to do with it is to pass it later to the fair_queue to identify a given class.
