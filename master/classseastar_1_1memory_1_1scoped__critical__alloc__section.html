seastar::memory::scoped_critical_alloc_section Class Reference

Detailed Description

Marks scopes that contain critical allocations.

Critical allocations are those, whose failure the application cannot tolerate. In a perfect world, there should be no such allocation, but we don't live in a perfect world. This information is used by other parts of the memory subsystem:

  • alloc_failure_injector will not inject errors into these scopes.
  • A memory diagnostics report will be dumped if an allocation fails in these scopes when the memory diagnostics subsystem is configured to dump reports for alloc_failure_kind alloc_failure_kind::critical or above. See set_dump_memory_diagnostics_on_alloc_failure_kind().
